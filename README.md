# CodeJump Photo Book Practice

CodeJumpの練習課題「Photo Book」をもとに制作したWebサイトです。

## Overview
シンプルなフォトブック形式のWebページを、HTMLとCSSで作成しました。  
レイアウトや余白、画像の見せ方を意識してコーディングしています。

## Technologies
- HTML
- CSS
